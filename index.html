My MCS dissertation research project was related mainly to text mining domain, purposed to learn to identify that whether or not two sentences in English language are paraphrases of each other. ParaDetect, a paraphrase identification system I developed using Java SE, which utilized machine learning classifier to learn to classify sentence pairs as paraphrases or not paraphrases of each other.

The system implemented achieved performance accuracy comparable to state-of-the-art paraphrase identification systems. Analysis of Microsoft Research Paraphrase Corpus, a standard dataset used mainly for evaluation of paraphrase identification techniques, and features based on semantic heuristics used for detecting not-paraphrase sentence pairs by ParaDetect were important part of the research and development. The research elements of this work were published under title “Paraphrase Identification using Semantic Heuristic Features” in Research Journal of Applied Sciences.
